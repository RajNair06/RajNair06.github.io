About

Behind the Code

I'm a backend engineer passionate about crafting production-grade systems that are reliable, observable, and built to scale. My focus is Python — FastAPI, Django, Flask — where I design REST APIs, real-time WebSocket backends, async data pipelines, and financial transaction systems with strong correctness guarantees.

My core stack spans PostgreSQL, Redis, Docker, Celery/Dramatiq, and JWT/OAuth2. I care deeply about clean architecture — idempotency, rate limiting, RBAC, and audit logging aren't afterthoughts, they're foundational. Every system I build is designed to be monitored, measured, and debugged in production.

Currently pursuing my B.Sc. in Information Technology at SMT CHM College, Ulhasnagar (2024–present), I spend my time building, breaking, and shipping real-world backends — from wallet engines to intrusion detection pipelines.
